# Docker Tutorial 

## Installing Docker on Ubuntu 

1. Follow instruction on docker 

2. Go to resolv.conf file 

```
sudo nano /etc/resolv.conf
```

3. Change nameserver to 8.8.8.8. Press ctrl + x and save the conf file
